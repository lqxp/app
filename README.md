# QXP Client - Electron App

An Electron application to access the QXP client without opening a browser.

## Installation

```bash
npm install
```

## Getting Started

Development mode:
```bash
npm start
```

With developer tools:
```bash
npm run dev
```

## Build

Create an executable for your platform:

```bash
# macOS
npm run build:mac

# Windows
npm run build:win

# Linux
npm run build:linux
```

## Features

- ✅ Direct access to QXP client
- ✅ Native Electron interface
- ✅ Fast startup
- ✅ Cross-platform support
